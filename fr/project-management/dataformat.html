Données raster¶

Les données raster peuvent rapidement devenir volumineuses si elles ne sont pas compressées, jusqu’à plusieurs Gb. Sur des appareils mobiles, ce n’est pas efficient.

Utiliser le format GeoPackage¶

Nous recommandons l’utilisation du format GeoPackage pour les données raster. Les commandes suivantes permettent de convertir un fichier nommé raster.tif vers un fichier raster.gpkg avec pyramides. Pensez à ajuter la projection EPSG:21781 (CH1903 LV03) avec la projection désirée.

gdal_translate --config OGR_SQLITE_SYNCHRONOUS OFF -co  APPEND_SUBDATASET=YES -co TILE_FORMAT=PNG_JPEG -a_srs EPSG:21781 -of GPKG raster.tif raster.gpkg
gdaladdo --config OGR_SQLITE_SYNCHRONOUS OFF -r AVERAGE raster.gpkg 2 4 8 16 32 64 128 256
