QField用户指南¶

打开项目¶

打开项目的操作为依次点击**菜单按钮” –> 选项按钮 –>打开项目**。

浏览并选择要打开的项目，或者使用**最近的项目**和**收藏夹**这两种快捷方式打开项目。

图例¶

打开**主菜单**，展开图层列表以显示地图图例。

识别要素¶

在地图上点击某要素可以识别它。 如果点击的位置上有多个要素（要么因为有多个要素确实彼此离的很近，要么因为地图中叠加有多个图层），它们都将显示在屏幕右侧的菜单列表中。

点击列表中的某个要素以访问其属性。

  1. 点击**左右箭头**滚动显示识别出的所有要素。
  2. 点击**居中按钮**使得选中的要素在地图上居中显示。
  3. 点击**编辑按钮**可以编辑选中要素的属性。
  4. 通过**子菜单**可以滚动访问所有的属性。

识别图层时的例外情况¶

通常情况下，不需要对所有的图层都进行查询。有些图层仅仅作为底图显示，这些图层的属性也不是关注的重点。

可以在QGIS桌面中管理图层列表，通过`项目 ‣ 项目属性 ‣ 识别图层`找到图层，取消其底图图层选项即可。

搜索¶

点击屏幕右上角的**搜索按钮**可以对图层进行搜索。

  1. 点击查询结果中的**名称**，可以将地图平移至要素所在位置（要素将以黄色高亮显示）。
  2. 点击**属性按钮**打开要素的属性信息。

切换地图主题¶

如果在QGIS中为项目定义了多个地图主题，可以通过**主菜单**中的第一个下拉列表项切换主题。

GPS定位¶

短按GPS按钮将打开GPS，获得位置信息后即将当前位置在地图上居中显示。

长按GPS按钮将显示“定位”菜单。

在定位菜单中，可以打开定位信息显示开关，显示当前坐标信息以及定位精度信息，坐标信息被自动转换为项目CRS。

注解

如果你看到的是WGS 84的经纬度坐标，而不是项目使用的CRS坐标值，很可能是因为还没有GPS信号。

使用外部GNSS接收器¶

可以使用独立的安卓应用程序为QField提供模拟位置信息。有很多的选择，其中之一是可以使用 `安卓NTRIP客户端<https://play.google.com/store/apps/details?id=com.lefebure.ntripclient>`_。

要使用此功能，您必须在Android设备上启用模拟位置 <https://www.youtube.com/watch?v=v1eRHmMiRJQ>。

绘制¶

如果要绘制新的要素，请在主菜单中将模式切换为 **编辑模式 **（*铅笔*按钮），选择需要添加新要素的图层（选中的图层将以灰色高亮显示）。

绘制点¶

将屏幕中心的十字瞄准线移动到目标位置，点击屏幕右下方的 加号按钮，创建一个新的点要素。

如果已经成功启用GPS，可以使用**保持居中**按钮使十字瞄准线保持居中状态。

绘制线和面¶

将屏幕中心的十字瞄准线移动待绘制线或面的起点，然后单击屏幕右下方的**加号(+)按钮**以添加第一个节点。

每点击一次**加号 (+)按钮 **就会添加一个新节点，通过连续添加节点构成线或面。

点击**减号(-)按钮**将删除最后一个添加的节点。

点击**关闭 (X)按钮**取消要素绘制。

当绘制了线要素的2个节点或面要素的3个节点后，将出现**保存按钮**，点击该按钮将完成绘制操作。

属性表单¶

几何对象绘制完成后，将显示属性表单，除非它被取消的了，提示用户录入新要素的属性信息。每个属性的右侧都有一个复选框，如果选中，系统会自动记住输入的属性值。当该图层再次添加新要素的时候，这些属性值将会被自动填充。

删除要素¶

删除要素只能在编辑模式下使用。

删除要素时，必须先选中它，然后点击垃圾桶图标。
