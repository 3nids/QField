Support and Development¶

Commercial Support¶

As developer of QField, OPENGIS.ch provides training packages and custom commercial support contracts with SLA

Development and Custom Apps¶

You probably have plenty of ideas for nice features that are not implemented yet. Please contact us to discuss the best approach to integrate this idea into QField or to create your own custom app.

Community Support¶

你可以再此提问 gis.stackexchange

你可以使用 user mailing list

基金¶

下面的团体帮助使QField变为了现实。我们热情地说：非常感谢！

信用¶

QField由opengis.ch开发，OPENGIS.ch负责开源软件的咨询、开发和培训，包括QField和QGIS。

API稳定性¶

QField目前尚不被认为是API稳定的。

许可¶

QField是根据GNU公共许可证（GPL）Version2或更高版本发布的。在此许可证下开发QField意味着您可以检查和修改源代码，并保证您始终可以访问基于QGIS的现场数据收集应用程序，该应用程序免费且可以自由修改。

您可以在此找到源码 on github
