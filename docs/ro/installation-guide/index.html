QField Installation Guide¶

Android Version¶

QField can safely be used with any version later than android 4.3.

Get it on GooglePlay¶

You can download the app directly from GooglePlay.

Obțineți un apk pre-compilat¶

De asemenea, oferim spre descărcare un apk pre-compilat, celor nu sunt în măsură sau nu doresc să utilizeze GooglePlay. Puteți obține qfield.apk din pagina versiunilor.
