Seleção de projeto¶

QField abre projectos QGIS no formato «.qgs». Atualmente não é possível brir ficheiros «.qgz».

QField usa um selecionador de ficheiros personalizado para abrir projetos. Ao escolher «Open project» no menu QField, o selecionador de ficheiros será abertos mostrando os seguintes locais:

  • The primary media/shared storage directory.
  • The application-specific directories on all shared/external storage devices where the application can place persistent files it owns if they are not already included in the primary storage directory.

Dentro dessas diretorias, apenas subdirectorias e ficheiros com a extensão «.qgs» são mostrados.

O QField exige que o arquivo do projeto esteja fisicamente no dispositivo, portanto, não é possível usar provedores que trabalham com arquivos remotos (por exemplo, Google Drive). Por outro lado, você pode usar aplicativos como syncthing ou nextcloud (ou dropbox e provavelmente muitos outros) que permitem a você sincronizar fisicamente o arquivo no seu dispositivo.

Directorias favoritas¶

Na primeira tela do seletor de arquivos, há uma seção mostrando os diretórios favoritos. Para adicionar um diretório aos favoritos, clique longo no seletor de arquivos no nome do diretório. Para remover uma entrada dos favoritos, clique na entrada na lista de favoritos.

Cartão SD externo¶

O seletor de arquivos mostra a raiz do cartão SD rotulada como «somente leitura» e o diretório específico do aplicativo separado do restante do cartão SD, pois é o único diretório ao qual o aplicativo tem acesso de gravação. O diretório é o seguinte:

/Android/data/ch.opengis.qfield/files

Nota

Tecnicamente, isso se deve ao fato de o Android tentar ACTION_OPEN_DOCUMENT e ACTION_OPEN_DOCUMENT_TREE, trabalhar e conceder acesso de leitura e gravação ao conteúdo do URI de retorno, e não ao arquivo vinculado real. Essa limitação faz sentido para a estrutura do Android, mas atualmente é inutilizável com o QField, pois o QField precisa acessar o arquivo real com um caminho real.

Atenção!¶

Ao armazenar dados nesta pasta, você corre o risco de perder dados. O diretório específico do aplicativo (/Android/data/ch.opengis.qfield/) será excluído pelo sistema se você desinstalar o QField do seu dispositivo Android. Isso é importante se você mantiver os dados no cartão SD externo.

Como tornar cartão SD externo em armazenamento interno¶

A partir do Android 6.0, você pode usar o cartão SD como armazenamento interno para o seu telefone Android. Um recurso chamado Adoptable Storage permite que o sistema operacional Android formate uma mídia de armazenamento externa como armazenamento interno permanente. Os dados no cartão SD adotado são criptografados e não podem ser montados em outro dispositivo.

Certifique-se de começar com um cartão SD vazio. Ele precisa estar vazio porque, quando o cartão é convertido em armazenamento «Interno», ele é formatado (apagado).

  1. Vá às «Definições» do dispositivo, e de seguida «Armazenamento».
  2. Seleccione o seu «Cartão SD», e de seguida toque no «three-dot menu» (canto superior direito), seleccione em «Definições» a partir daí.
  3. Agora seleccione «Formatar como interno», e depois «Apagar & Formatar».
  4. O seu cartão SD será formatado como armazenamento interno.
  5. Reinicie o seu telefone.

Se não reiniciar o seu telefone, muitas coisas podem não correr de forma correcta, por isso, certifique-se de o fazer.

Pode ser possível que o recurso Adoptable Storage não esteja presente no seu dispositivo, mesmo que esteja executando o Android 6.0 e superior. O fabricante do dispositivo pode ter desativado o recurso. No entanto, existem métodos de linha de comando que permitem forçar um dispositivo a adotar um armazenamento.
