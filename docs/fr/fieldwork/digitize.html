Collect features¶

Pour commencer la vectorisation de nouvelles entités, il faut activer le mode Édition dans le volet de gauche (bouton crayon) et sélectionner la couche sur laquelle vous souhaitez ajouter de nouvelles entités (la couche sera surinée en gris).

Points¶

Déplacer la carte de sorte que la position désirée se trouve sous le réticule situé au centre de l’écran et cliquer sur sur le bouton + dans le coin inférieur droit pour confirmer la création de cette nouvelle entité.

Si vous avez activé la localisation, vous pouvez utiliser le bouton Centrer pour forcer le centrage de votre position sous le réticule.

Lignes et polygones¶

Déplacer la carte de sorte que le point de départ désiré de la ligne ou du polygone se trouve sous le réticule situé au centre de l’écran. Cliquer ensuite sur le bouton (+) dans le coin inférieur droit pour ajouter le premier nœud.

Procéder de la même manière et répéter l’opération pour former votre ligne ou votre polygone en cliquant sur le bouton (+) chaque fois que vous souhaitez ajouter un nouveau nœud.

Cliquer sur le bouton (-) pour supprimer le dernier nœud ajouté.

Cliquer sur le bouton (X) pour annuler complètement la création de l’entité.

Quand vous avez ajouté au moins 2 nœuds pour une ligne ou 3 nœuds pour un polygone, un bouton Sauvegarder apparaît. Cliquer dessus pour terminer votre dessin et enregistrer l’entité.

Formulaire d’attributs¶

Après l’ajout d’une géométrie et s’il existe, le formulaire d’attributs s’affiche et l’utilisateur est invité à saisir les attributs de la nouvelle entité. Le formulaire qui apparaît permet d’entrer les valeurs des attributs de la nouvelle entité. Les cases à cocher à droite de chaque attribut permettent de mémoriser chaque attribut individuellement. La prochaine fois que vous ajouterez une entité sur la même couche, ces attributs seront déjà pré-remplis.

Delete features¶

La suppression d’entités est uniquement possible en mode Édition.

To delete a feature, identify it first by a short tap on it. Then select it with a long press in the identification list, select further features to delete if you want and click the trash button to confirm the deletion.

Edit features¶

To edit, tap on Menu button -> pencil.

You can edit an existing feature by moving, adding or deleting the vertex and also spliting.

Vertex tool¶

It is possible to move and delete vertices of geometries and add new ones.

Split tool¶

It is possible to split an existing geometry by drawing a line where it should be split.
