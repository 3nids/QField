Géolocalisation¶

Un appui court sur le bouton du GPS l’active et centre la carte sur la position actuelle, dès que les données de géolocalisation sont disponibles.

Un appui long sur le bouton du GPS affiche le menu de géolocalisation.

Dans le menu de géolocalisation, l’affichage des données de position peut être activé. Les coordonnées affichées sont reprojetées dans le SCR du projet.

Note

Si les informations sont affichées en WGS84 au lieu de votre propre SCR, c’est probablement parce qu’aucun signal n’est disponible.

Utilisation d’un récepteur GNSS externe¶

It is possible to provide a mock location via a separate android app to QField. There are several options for this, one of them is Android NTRIP Client.

Pour l’utiliser vous devez autoriser votre smartphones à utiliser les positions fictives <https://www.youtube.com/watch?v=v1eRHmMiRJQ>`_.
