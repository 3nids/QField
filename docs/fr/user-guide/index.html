Guide d’utilisateur QField¶

Ouvrir un projet¶

Pour ouvrir un projet, cliquer sur le bouton Menu puis Option > Ouvrir un projet.

Naviguer dans l’arborescence de l’appareil puis sélectionner le projet souhaité ou utiliser les raccourcis Projets récents et Dossiers favoris.

Légende de carte¶

Ouvrir le menu et dérouler la liste des couches pour afficher la légende de la carte.

Sélectionner des entités¶

Cliquer sur une entité de la carte pour la sélectionner. Si d’autres entités sont proches de cette dernière (soit parce qu’il y a plusieurs entités très proches les unes des autres ou parce que plusieurs couches se chevauchent), elles seront toutes sélectionnées et listées dans le volet à droite de l’écran.

Cliquer sur l’une d’entre elles dans la liste pour accéder à ses attributs.

  1. Utiliser les flèches pour faire défiler les entités sélectionnées.
  2. Cliquer sur le bouton centrer pour centrer la carte sur l’entité sélectionnée.
  3. Cliquer sur le bouton éditer pour modifier les attributs de l’entité sélectionnée.
  4. Faire défiler les onglets pour accéder à l’ensemble des attributs.

Exceptions couches interrogeables¶

Il n’est souvent pas nécessaire de pouvoir interroger chacune des couches du projet. Certaines couches sont uniquement utiles comme fond de plan et leurs attributs n’ont pas d’intérêt.

Il est possible de gérer la liste des couches interrogeables dans QGIS desktop Projet ‣ Propriétés du projet ‣ Identifier les couches . Décocher à cet endroit les couches qui ne doivent pas être interrogeables.

Rechercher¶

Cliquer sur le bouton Rechercher dans le coin supérieur droit pour rechercher dans les couches.

  1. Cliquer sur le nom affiché dans les résultats pour centrer la carte sur l’entité (cette dernière sera par ailleurs colorée en jaune).
  2. Cliquer sur le bouton Attributs pour ouvrir directement les attributs de l’entité.

Changer le thème de la carte¶

Si vous avez défini des thèmes de carte dans votre projet QGIS, vous pouvez passer de l’un à l’autre en les sélectionnant grâce au menu déroulant dans le volet de gauche.

Géolocalisation¶

Un appui court sur le bouton du GPS l’active et centre la carte sur la position actuelle, dès que les données de géolocalisation sont disponibles.

Un appui long sur le bouton du GPS affiche le menu de géolocalisation.

Dans le menu de géolocalisation, l’affichage des données de position peut être activé. Les coordonnées affichées sont reprojetées dans le SCR du projet.

Note

Si les informations sont affichées en WGS84 au lieu de votre propre SCR, c’est probablement parce qu’aucun signal n’est disponible.

Utilisation d’un récepteur GNSS externe¶

Il est possible de générer un emplacement fictif grâce à une application Android tierce pour l’utiliser dans QField. Plusieurs solutions sont possibles ; l’une d’elles est l’application Android NTRIP Client.

Pour l’utiliser vous devez autoriser votre smartphones à utiliser les positions fictives <https://www.youtube.com/watch?v=v1eRHmMiRJQ>`_.

Vectoriser¶

Pour commencer la vectorisation de nouvelles entités, il faut activer le mode Édition dans le volet de gauche (bouton crayon) et sélectionner la couche sur laquelle vous souhaitez ajouter de nouvelles entités (la couche sera surinée en gris).

Points¶

Déplacer la carte de sorte que la position désirée se trouve sous le réticule situé au centre de l’écran et cliquer sur sur le bouton + dans le coin inférieur droit pour confirmer la création de cette nouvelle entité.

Si vous avez activé la localisation, vous pouvez utiliser le bouton Centrer pour forcer le centrage de votre position sous le réticule.

Lignes et polygones¶

Déplacer la carte de sorte que le point de départ désiré de la ligne ou du polygone se trouve sous le réticule situé au centre de l’écran. Cliquer ensuite sur le bouton (+) dans le coin inférieur droit pour ajouter le premier nœud.

Procéder de la même manière et répéter l’opération pour former votre ligne ou votre polygone en cliquant sur le bouton (+) chaque fois que vous souhaitez ajouter un nouveau nœud.

Cliquer sur le bouton (-) pour supprimer le dernier nœud ajouté.

Cliquer sur le bouton (X) pour annuler complètement la création de l’entité.

Quand vous avez ajouté au moins 2 nœuds pour une ligne ou 3 nœuds pour un polygone, un bouton Sauvegarder apparaît. Cliquer dessus pour terminer votre dessin et enregistrer l’entité.

Formulaire d’attributs¶

Après l’ajout d’une géométrie et s’il existe, le formulaire d’attributs s’affiche et l’utilisateur est invité à saisir les attributs de la nouvelle entité. Le formulaire qui apparaît permet d’entrer les valeurs des attributs de la nouvelle entité. Les cases à cocher à droite de chaque attribut permettent de mémoriser chaque attribut individuellement. La prochaine fois que vous ajouterez une entité sur la même couche, ces attributs seront déjà pré-remplis.

Supprimer des entités¶

La suppression d’entités est uniquement possible en mode Édition.

Pour supprimer une entité, la sélectionner d’abord, puis appuyer sur le bouton Corbeille.
