Collect features¶

Para comenzar a digitalizar nuevos objetos, active el Modo edición en el menú principal (botón lápiz) y seleccione la capa a la que quiere añadir objetos nuevos (la capa se resaltará en gris).

Puntos¶

Desplace la cruz del centro de la pantalla a la localización deseada y pulse el botón + de la esquina inferior derecha de la pantalla para confirmar la creación del nuevo objeto de punto.

Puede usar el botón Mantener centrado para forzar que la cruz se centre en su localización si tiene activado el GPS.

Líneas y polígonos¶

Desplace la cruz del centro de la pantalla a la localización deseada de inicio de la línea o polígono y pulse el Botón Más (+) de la esquina inferior derecha de la pantalla para añadir el primer nodo.

Continúe añadiendo puntos para formar su línea o polígono pulsando el Botón Más (+) cada vez que quiera añadir un nuevo nodo.

Pulse el Botón Menos (-) para eliminar el último nodo añadido.

Pulse el Botón Cerrar (X) para cancelar por completo la creación del objeto.

Cuando haya añadido al menos 2 nodos para una línea o 3 para un polígono, aparecerá un Botón Guardar. Púlselo para finalizar su diseño.

Formulario de atributos¶

Después de digitalizar una geometría, se mostrará el formulario de atributos (si no está suprimido) y se pedirá al usuario que introduzca los atributos del nuevo objeto. El formulario que aparece le permite introducir valores de atributos para cada nuevo objeto. Las casillas de verificación a la derecha de cada atributo permiten recordar cada atributo individualmente. La próxima vez que añada un objeto a la misma capa estos atributos estarán ya preintroducidos.

Delete features¶

La eliminación de elementos sólo es posible en el modo de digitalización.

To delete a feature, identify it first by a short tap on it. Then select it with a long press in the identification list, select further features to delete if you want and click the trash button to confirm the deletion.

Edit features¶

To edit, tap on Menu button -> pencil.

You can edit an existing feature by moving, adding or deleting the vertex and also spliting.

Vertex tool¶

It is possible to move and delete vertices of geometries and add new ones.

Split tool¶

It is possible to split an existing geometry by drawing a line where it should be split.
