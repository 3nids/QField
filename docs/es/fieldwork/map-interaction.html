Interact with the map¶

Leyenda del mapa¶

Abra el Menú principal y expanda la lista de capas para mostrar la leyenda del mapa.

Identificar objetos¶

Pulse en un objeto del mapa para identificarlo. Si hay varios objetos en el punto donde ha pulsado (bien porque realmente hay varios objetos cerca unos de otros o porque hay varias capas superpuestas) todos ellos se mostrarán en el menú que se abre a la derecha de la pantalla.

Pulse en uno de los objetos listados para acceder a sus atributos.

  1. Pulse las flechas para navegar por todos los objetos identificados.
  2. Pulse el botón centrar* para centrar el mapa en el objeto seleccionado.
  3. Pulse el botón editar para editar los atributos del objeto seleccionado.
  4. Scroll through the sub-menus to access all the attributes.

Excepciones a la identificación de capas¶

A menudo no es necesario que todas las capas puedan consultarse. Algunas capas solo se presentan como base y sus atributos no son de interés.

Puede gestionar estas capas en QGIS Desktop en Proyecto ‣ Propiedades del Proyecto ‣ Fuentes de datos y desmarcar las capas base en la columna Identificable.
